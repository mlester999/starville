# Phase 7 cozy systems

The Phase 7 loop is deliberately small and complete:

1. Enter through the existing `$STAR` access boundary and receive exactly one starter grant.
2. Buy one of three seed types from the fixed-price Lantern General Store.
3. Plant and water one of six private plots; the database clock controls growth.
4. Harvest the deterministic yield into available inventory space.
5. Sell eligible goods or use four cooking and two crafting recipes.
6. Enter the private starter cottage and place, move, rotate, or remove owned furniture.

The starter content uses Moonbean, Sunroot, and Cloudberry crops; four simple cooked foods; Garden
Twine and a Willow Chair crafting recipe; and six furniture definitions. Names, quantities, prices,
growth durations, yields, recipe inputs, and outputs are configuration rather than client rules.

There is no stamina, hunger, health, combat, durability, randomized quality, paid boost, multiplayer
farm, home visiting, trading, gifting, auction, quest, or reward system in this phase.

The desktop UI keeps WASD movement and Shift jogging. `E` activates the nearest valid typed
interaction. Slots 1–8 select the quickbar only when focus is not inside a text-entry control.
Inventory and action dialogs block Phaser input and restore focus on close. Unsupported phones keep
the existing keyboard-required notice; Phase 7 does not claim touch gameplay support.
